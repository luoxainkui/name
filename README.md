# newbie
>>>>这是我第一个基于Python开发的小项目，支持管理员模式和学生模式
>>>>项目介绍:
*   **图书管理**：增删改查图书信息、修改图书状态。
*   **学生管理**：学生信息的增删改查。
*   **借阅归还**：学生凭学号借阅/归还图书，并自动修改书籍状态。
*   **日志系统**：详细记录每一次借阅、归还的时间、状态和学生信息。

>>>>项目结构:
[text](core/Library_books.py) =="管理员图书"
[text](core/logs.py) == "日志"
[text](core/students.py) == "管理员学生模式"
[text](main.py) == "接口,里面进行了分流支持学生视角"
